# Panduan Pengguna
## Customer
1. Buka aplikasi
   - Jalankan server lokal (XAMPP atau MAMP)
   - Buka browser dan akses http://localhost/PROJECT AKHIR TECHART HOTEL
2. Registrasi & Login
   - Registrasi: mengisi regis form nama, email, dan password
   - Login: masukkan email dan password
3. Gunakan fitur sesuai role
   - Melihat halaman utama (home page)
   - Melihat informasi hotel (about page)
   - Menghubungi pihak hotel (contact page)
   - Melihat list kamar hotel (hotel page)
   - Melihat detail kamar hotel (detail page)
   - Melakukan booking kamar: mengisi form nama, email, tipe kamar, jumlah tamu, tgl check in/out (confirm booking)
   - Melakukan pembayaran (payment)

## Admin
1. Login
   - Akses backend admin http://localhost/PROJECT AKHIR TECHART HOTEL/admin
   - Masukkan username & password admin
2. Kelola data sistem
   - Mengelola kamar hotel: tambah, edit, hapus
   - Mengelola booking kamar hotel: konfirmasi atau mengubah status booking
   - Mengelola data customer
3. Logout setelah selesai
