##  Deskripsi Project
Website Buddies Hotel merupakan platform pemesanan kamar hotel berbasis web yang dirancang untuk mempermudah pengunjung dalam melihat informasi detail kamar, melakukan booking, serta menyelesaikan pembayaran secara online. Website ini bertujuan memberikan pengalaman pemesanan yang lebih praktis tanpa harus datang langsung ke hotel, sekaligus membantu pihak hotel dalam mengelola data kamar, pengguna, dan transaksi secara lebih efisien. Dengan adanya sistem ini, proses reservasi menjadi lebih cepat, terstruktur, dan mudah digunakan bagi customer maupun admin.

## Teknologi yang Digunakan
- Backend: Native PHP
- Frontend: HTML5, CSS3, JavaScript
- Database: MySQL
- Server Lokal: XAMPP / MAMP

##  Cara Menjalankan Project
1. Clone repository:
   ```bash
   git clone https://github.com/JesslyneZhang/project-akhir-techart.git
   ```
2. Pindahkan project ke ke htdocs (XAMPP).
   /Applications/XAMPP/htdocs/
3. Import database dari /docs/db.sql.
4. Jalankan di browser: http://localhost/BuddiesHotel/
